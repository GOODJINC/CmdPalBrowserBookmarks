# 개인정보처리방침

[English](PRIVACY.md) | 한국어

Browser Bookmarks for Command Palette는 Microsoft PowerToys Command Palette에서 브라우저 북마크를 검색하고 열 수 있도록 도와주는 확장입니다.

## 데이터 접근

이 확장은 사용자가 확장 설정에서 활성화한 브라우저와 프로필에 따라, 사용자 기기에 저장된 브라우저 프로필 파일에서 북마크 데이터를 읽습니다.

지원되는 데이터 원본은 다음을 포함할 수 있습니다.

- Microsoft Edge 북마크 파일
- Google Chrome 북마크 파일
- Mozilla Firefox 북마크 데이터베이스

확장은 Command Palette 안에서 검색 결과를 표시하기 위해 북마크 제목, URL, 브라우저 이름, 프로필 이름, 북마크 폴더 경로를 읽을 수 있습니다.

## 로컬 처리

북마크 데이터는 사용자 기기에서 로컬로 처리됩니다.

이 확장은 사용자의 북마크 데이터를 개발자 또는 외부 서버로 업로드, 판매, 공유, 전송하지 않습니다.

## 설정

이 확장은 활성화한 브라우저, 선택한 프로필, 검색 옵션, 언어, 브라우저 실행 방식 같은 자체 설정을 로컬에 저장합니다.

이 설정은 확장 기능을 동작시키는 용도로만 사용됩니다.

## 네트워크 접근

이 확장은 분석 데이터를 수집하거나 개인정보를 전송하기 위해 네트워크 접근을 사용하지 않습니다.

사용자가 북마크를 열면 선택한 웹사이트는 브라우저에서 로드됩니다. 웹사이트를 연 이후의 데이터 교환은 이 확장이 아니라 사용자의 브라우저와 해당 웹사이트에서 처리합니다.

## 제3자 서비스

이 확장은 제3자 분석, 광고, 추적 서비스와 연동하지 않습니다.
