# Privacy Policy

English | [한국어](PRIVACY.ko.md)

Browser Bookmarks for Command Palette is a Microsoft PowerToys Command Palette extension that helps you search and open browser bookmarks.

## Data Access

This extension reads bookmark data from browser profile files on your device, depending on the browsers and profiles you enable in the extension settings.

Supported sources may include:

- Microsoft Edge bookmark files
- Google Chrome bookmark files
- Mozilla Firefox bookmark databases

The extension may read bookmark titles, URLs, browser names, profile names, and bookmark folder paths so it can show search results inside Command Palette.

## Local Processing

Bookmark data is processed locally on your device.

This extension does not upload, sell, share, or transmit your bookmark data to the developer or to any external server.

## Settings

The extension stores its own settings locally, such as enabled browsers, selected profiles, search options, language, and browser launch preference.

These settings are used only to operate the extension.

## Network Access

This extension does not use network access to collect analytics or send personal data.

When you open a bookmark, your browser loads the selected website. Any data exchanged after opening the website is handled by your browser and that website, not by this extension.

## Third-Party Services

This extension does not integrate with third-party analytics, advertising, or tracking services.
